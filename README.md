# Caltrack

A modern, mobile-friendly calorie tracking web app. Track your daily calorie intake, view history, and monitor nutritional stats. Built with React (frontend) and Node.js/Express (backend).

## Features
- Log daily calorie intake
- View intake history
- Track nutritional stats
- Basic authentication
- Mobile-friendly design

## Getting Started

### Backend
1. Go to the `backend` folder
2. Run `npm install`
3. Start the server: `npm start`

### Frontend
1. Go to the `frontend` folder
2. Run `npm install`
3. Start the app: `npm start`

---

This is an MVP. Data is stored locally (SQLite or JSON file).
